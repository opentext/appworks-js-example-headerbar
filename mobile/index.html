Header Button Demo

You have a few buttons buttons below which set and reset the header buttons (the ones up there ^^ on the left and right).

You can also mask and unmask the header to fit in with your UI.

"Set Header Buttons" is invoked when the page loads, and changes the buttons

"Set Hamburger Button" sets the second left button the menu icon

"Set Text Buttons" sets the left and right buttons to text

"Reset Header Buttons" sets all buttons to their default state
